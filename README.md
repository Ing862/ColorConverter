# ColorConverter

## Opis
Aplikacja moblina na urządzenia android do konwersji dwóch modeli barw: RGB oraz CMYK. Aplikacja zrealizowana na potrzeby zajęć Grafika Komputerowa.

## Działanie
Użytkownik może podać wartości modeli barw na dwa sposoby - zarówno poprzez pole tekstowe, jak również suwaki umieszczone pod każdą z liter konkretnego modelu. Wraz ze zmianą jednej wartości następuje przeliczanie pozostałych i zakualizowanie ich w interfejsie. W polu poniżej wyśwetlany jest kolor odpowiadający modelom barw.
